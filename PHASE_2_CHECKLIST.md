escalated\n  ↓\n  → rejected\n```\n\n- **open**: Student just reported, waiting for admin/tech assignment\n- **assigned**: Admin assigned to technician, tech notified\n- **in_progress**: Technician started working on it\n- **resolved**: Tech finished, student notified, asset status updated to \"working\"\n- **escalated**: Tech needs help (manual admin action)\n- **rejected**: Not a real issue (manual decision)\n\n## 🚀 Quick Test Flow\n\n1. **Setup Admin Account** (from Phase 1)\n   ```\n   Email: admin@test.com\n   Password: TestPass123\n   Role: Admin\n   ```\n\n2. **Create Student
